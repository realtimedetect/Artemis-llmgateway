# Recording Guide for llm_gateway Demo Video

## Goal
Produce a 4 to 5 minute product video explaining features, differentiators, and benefits.

## Assets to Use
- Script: docs/video/llm_gateway_video_script.md
- Storyboard: docs/video/llm_gateway_storyboard.md
- Subtitles: docs/video/llm_gateway_subtitles.srt

## Fastest Workflow (OBS + Browser)
1. Start llm_gateway with sample data.
2. Open dashboard pages in browser tabs:
   - Overview
   - Providers
   - Routes
   - API Keys
   - Costs
   - Cache
   - Audits
3. Set browser zoom to 110%.
4. In OBS, capture browser window at 1080p, 30 fps.
5. Read from script and follow storyboard scene timing.
6. Export recording as MP4 (H.264, 8 to 12 Mbps).
7. Import MP4 into CapCut or DaVinci Resolve and add SRT subtitles.

## Production Tips
- Keep each dashboard page visible for at least 8 to 12 seconds.
- Highlight only one feature per scene to avoid visual overload.
- Avoid rapid scrolling.
- Use one consistent voice tone and speed.
- Keep background music low and optional.

## Suggested Intro and Outro Overlay Text
Intro:
- llm_gateway
- Multi-provider LLM control plane

Outro:
- One integration pattern
- Better reliability and governance
- Better cost visibility and control
